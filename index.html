Project's Guidelines

  1. Books are stored in a simple array.
  2. Create a Book constructor object that takes user input.
  3. Each book should be displayed in the page in a table or as 'cards'.
  4. There is a 'Add new Book' button that brings up a form to input data.
  5. There is a button on each book display to delete it from the list.
  6. There is a button to change read status of each book.

Optional / Bonus

  • Add object permanence by learning to use localStorage, so the webpage remembers the books added / deleted.
